> Aplikacja do dbania o Twój wzrok stworzona we współpracy z KN Visus!

## 📋 O projekcie

Celem projektu jest stworzenie lekkiej aplikacji webowej, która pomoże programistom oraz osobom pracującym przy komputerze dbać o zdrowie oczu. Narzędzie będzie przypominać o regularnych przerwach i dostarczać cenne informacje o higienie wzroku, bazując na wiedzy optometrycznej.


## 🛠 Technologie

Wymień główne języki i biblioteki.

* **Język:** TypeScript
* **Biblioteki:** React, Tailwind, Shadcn
* **Narzędzia:** PostreSQL

## 💻 Jak uruchomić (Getting Started)

Instrukcja krok po kroku, jak uruchomić projekt lokalnie.

### Wymagania wstępne
* `git`
* `node >= v24.5.0`
* `python` (tylko do instalacji `pre-commit`)

### Instalacja

1. Sklonuj repozytorium:
   ```bash
   git clone https://github.com/KN-Algo/AlgoVisus.git
   ```

2.  Zainstaluj zależności frontendowe:
    ```bash
    npm install
    ```

3. Zainstaluj zależności wymagane do funkcjonowania pre-commit:
    ```bash
    pip install pre-commit
    ```

4. Zainstaluj pre-commit w repozytorium:
    ```bash
    pre-commit install
    ```

### Uruchomienie

Komenda do uruchomienia programu:

```bash
npm run dev
```
